User stories
------------
* Simulate a single game
  + specify ruleset
  + click start
  + view result
* Simulate a session
  + specify:
    - ruleset
    - bankroll
    - betting strategy
    - simulation length
  + click start
  + view result
* Simulate multi-session
  + specify:
    - ruleset
    - wallet size
    - initial bankroll
    - stop loss
    - profit target
    - betting strategy
    - simulation length
  + click start
  + view result
